성인용품 리뷰 서비스로, 거부감 없이 편하게 사용할 수 있도록 만든 어플리케이션입니다.



## Getting Started 📖

1. git clone

   ```cmd
   $ git clone https://github.com/prography/snowflake-backend
   ```

2. set env variables
    ```
    SECRET_KEY=
    AWS_ACCESS_KEY_ID=
    AWS_SECRET_ACCESS_KEY=
    AWS_STORAGE_BUCKET_NAME=
    ```
